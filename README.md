# Frontend Mentor Assessment: HTML, CSS, JS & GitHub Basics

Welcome to your practical assessment! This assignment is designed to evaluate your fundamental frontend development skills (HTML5, CSS3, Vanilla JavaScript) and your proficiency with version control using GitHub. 

You will be completing two separate challenges from Frontend Mentor.

---

## 🚀 The Challenges

### 1. Time Tracking Dashboard

* **Objective:** Build a responsive dashboard layout that dynamically switches between daily, weekly, and monthly data views.
* **Challenge Link:** [Frontend Mentor - Time Tracking Dashboard](https://www.frontendmentor.io/challenges/time-tracking-dashboard-UIQ7167Jw)

### 2. Tip Calculator App

* **Objective:** Build a functional tip calculator that handles numeric inputs, custom percentage math, validation errors, and dynamic state resets.
* **Challenge Link:** [Frontend Mentor - Tip Calculator App](https://www.frontendmentor.io/challenges/tip-calculator-app-ugJNGbJUX)

---

## 🎯 Evaluation Criteria

Your submissions will be reviewed based on the following metrics:

| Skill Area | What We Are Looking For |
| :--- | :--- |
| **HTML5** | Correct use of semantic tags (`<main>`, `<section>`, `<button>`, proper labels, and input types). Structural accessibility (ARIA attributes where necessary). |
| **CSS3** | Layout accuracy (Flexbox/Grid), full responsiveness (mobile-first approach preferred), proper hover/active/focus interaction states. |
| **JavaScript** | Bug-free DOM manipulation, clean event handling, robust state or form validation logic, and meaningful variable naming. |
| **Git & GitHub** | Follow a clean branching workflow. Clear, atomic commit history detailing structural updates. Well-structured repository. |

---

## ⏰ Submission Guidelines

* Once completed, reply directly with the links to your two public GitHub repositories.
* Make sure your final code is pushed to the default branch before sharing.

Good luck! Take your time to review the style guides, structure your code carefully, and make sure your Git commit history tells a clear story of how you built the project.
